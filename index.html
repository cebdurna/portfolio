Projects

Arduino Cotton LED Panel

This Arduino Micro project utilizes LEDs and a cotton pad to create a display that cycles different display options with the press of a button.

15 LEDs are placed on a breadboard in 5x3 configuration with each LED mapped to a seperate output pin on the Arduino to ensure individual control. A cotton pad is placed on top of the LEDs to diffuse the light and present only the LEDs that are lit. Finally, a button is wired as input to control different display cycles which can be configured by modifying the attached source code.

Arduino Midi Box

This is a music box that utilizes the MIDI interface to create/manipulate sounds. The project utilizes RFID scanners, a photoresistor and buttons to detect user actions and map them to sounds.

Two different RFID scanners placed underneath the surface of this MIDI music box detect different tokens placed on top. These tokens triger different audio tracks as a result. A photoresistor placed on the surface serves a similar purpose by detecting shadows cast on the surface. Finally, three buttons placed on the music box enable a user to create different notes of whistles.

Physical Bar Graph

This is a physical bar graph that utilize stepper motors and an arduino micro controller to translate data into physical bars.

Three stepper motors are mounted on bars with the use of 3d printed mounting brackets, rack and pinions. These stepper motors are controlled with the use of a microcontroller to convert data into motor steps that are translated into vertical bar movements.

Pulse Balloons

The project collects pulse data of the user and reflects the current user's cardiovascular state by inflating corresponding painted balloons.

For pulse sensing, a MAX30102 sensor was attached to a velcro wrist strap a user can wear. An air pump in combination with multiple air valves can be used to inflate and deflate balloons, but due to better availability of parts, this project uses three distinct air pumps for three balloons. The pumps are supplied with 5V and are controlled with MOSFETs. An Arduino microcontroller handles the logic, pulse data and MOSFET gates.
